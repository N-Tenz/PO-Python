Het begin moeilijk en was erg langdradig. Dat was doordat wij gebruikt gingen maken van pygame voor onze video game, daardoor was minstens de helft van de code nieuw terrein voor ons. Dit bestreden we door pygame.org of youtube filmpjes te bekijken voor oplossingen van onze problemen, dat werd nog moeilijker doordat de meerderheid van de video's rare modules bevat en niet goed uitlegd. Ik heb veel chatGPT moeten gebruiken om modules te kunnen importeren. Een youtube filmpje werkt daar niet bij het is veel gemakkelijker om een zoekmachine te gebruiken die meteen reactie geeft. In de opdracht heb ik de start menu gemaakt. Als je de code runned dan kom je meteen bij het startscherm, als je dan op start drukt dan kom je bij een scherm waar je je gebruikersnaam invult, daarna kom je in het spel. Als je inplaats daarvan op exit drukt dan komt er eerst een pop up scherm dat ervoor zorgt dat je niet perongelijk weggaat. Daarnaast heb ik vooral deze html pagina gemaakt. Ik vond deze ervaring erg leuk, want ik heb altijd al videogames willen maken en nu weet ik hoe ik het moet doen, naast alle moeizaamheden. Ik heb tijdens deze opdracht leren te werken met pygame codes, maar daarnaast heb ik mijn kennis bij het gebruiken en maken van functies, variabelen for- en while- loops bevorderd.
